# Cybersecurity Portfolio – Jason Power

A structured GitHub portfolio for senior-level cybersecurity work. This repository is organized to highlight practical experience in detection engineering, cloud security, automation, threat hunting, and security architecture.

## Portfolio Goals
- Demonstrate hands-on, portfolio-ready cybersecurity work
- Show senior-level thinking through documentation, repeatable processes, and architecture decisions
- Create artifacts that support resume bullets, interviews, and job applications

## Recommended Workflow
1. Pick one project folder
2. Read the local README
3. Run or extend the starter artifact
4. Document your findings
5. Add metrics, screenshots, and lessons learned
